# Tic-Tac-Toe Web Application

A simple and interactive Tic-Tac-Toe game developed using HTML, CSS, and JavaScript.

## Features

- Two player game (Player X and Player O)
- Displays current player turn
- Checks winning conditions
- Shows draw result
- Score board for Player X, Player O, and Draw
- Restart game button
- Responsive and attractive user interface

## Technologies Used

- HTML
- CSS
- JavaScript
